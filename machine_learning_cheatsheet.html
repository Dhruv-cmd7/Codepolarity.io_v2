Introduction to Machine Learning

Intro

Machine Learning (ML) is a field of AI that enables computers to learn from data and make predictions or decisions without being explicitly programmed. ML powers applications like recommendation systems, image recognition, and language translation.

  • Automates pattern recognition and prediction
  • Uses data to improve performance over time
  • Key for AI, data science, and analytics

Types of Machine Learning

Types
  • Supervised Learning: Learn from labeled data (e.g., regression, classification)
  • Unsupervised Learning: Find patterns in unlabeled data (e.g., clustering, association)
  • Reinforcement Learning: Learn by trial and error, using rewards (e.g., game AI, robotics)
  • Semi-supervised & Self-supervised: Use both labeled and unlabeled data

Bias-Variance Tradeoff

Tradeoff
  • Bias: model's assumptions (underfitting)
  • Variance: sensitivity to training data (overfitting)
  • Goal: find optimal balance
  • Regularization helps control variance
# High bias: Linear model on non-linear data
# High variance: Complex model on small dataset
# Solution: Regularization
from sklearn.linear_model import Ridge
ridge = Ridge(alpha=1.0)  # Controls variance

Activation Functions

Activation
  • ReLU: max(0, x) - most common
  • Sigmoid: 1/(1+e^(-x)) - for probabilities
  • Tanh: (e^x - e^(-x))/(e^x + e^(-x))
  • Softmax: for multi-class classification
import numpy as np
def relu(x): return np.maximum(0, x)
def sigmoid(x): return 1 / (1 + np.exp(-x))
def softmax(x): return np.exp(x) / np.sum(np.exp(x))

Backpropagation

Backprop

  • Chain rule for derivatives

Q-Learning & MDPs

Q-Learning
  • Markov Decision Processes (MDPs)
  • Q-table: state-action value function
  • Q-learning update rule
  • Epsilon-greedy exploration
import numpy as np
Q = np.zeros((n_states, n_actions))
alpha = 0.1  # learning rate
gamma = 0.9  # discount factor
Q[state, action] = Q[state, action] + alpha * (reward + gamma * max(Q[next_state]) - Q[state, action])

Complexity Analysis

Complexity
Algorithm Time Space
Linear Regression O(n²d) O(d)
Logistic Regression O(n²d) O(d)
Decision Tree O(n log n) O(n)
Random Forest O(t × n log n) O(t × n)
SVM O(n²) to O(n³) O(n²)
K-Means O(nkd) O(n + kd)
Neural Network O(n × layers) O(parameters)

n = samples, d = features, k = clusters, t = trees

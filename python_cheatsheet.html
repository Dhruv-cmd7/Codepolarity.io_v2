Slicing

Slicing 
lst = [0, 1, 2, 3, 4]
print(lst[1:4])    # [1, 2, 3]
print(lst[-2:])    # [3, 4]
print(lst[::-1])   # [4, 3, 2, 1, 0]

List Comprehensions

Comprehensions

evens = [x for x in range(10) if x % 2 == 0]
